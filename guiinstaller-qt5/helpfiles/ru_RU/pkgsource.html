Базовые настройки

Здесь вы можете выбрать главный источник установки, а также включить обновление системы во время установки

Источник установки

Здесь вы можете выбрать основной источник установки:

  • Установка с диска (DVD или флешка): установка с носителя, с которого вы загрузились. Если вы загрузились с флешки с AgiliaLinux, то для установки с флешки выбирайте этот вариант
  • Установка из Интернета: установка по сети. Загружает все последние доступные пакеты из официального репозитория core32.agilialinux.ru или core64.agilialinux.ru (зависит от архитектуры)
  • ISO-образ диска: установка с ISO-образа, расположенного на жёстком диске. При выборе этого варианта перед следующим шагом вам надо будет указать путь к ISO-образу. Будьте в дальнейшем внимательны: раздел, на котором лежит ISO-образ, нельзя будет отформатировать при установке!
  • Директория на HDD: установка из папки с пакетами на жёстком диске. При выборе этого варианта перед следующим шагом вам нужно будет указать путь к папке с пакетами
  • Другой URL: установка из собственного репозитория, доступного по URL.

Расширенные настройки

Автоматическое обновление

Вы можете установить галочку "Автоматически обновить пакеты из интернет-репозитория", чтобы получить свежие версии пакетов из Интернета, устанавливаясь с локальных носителей (DVD или ISO). Итоговый результат будет идентичен выбору пункта "Официальный сетевой репозиторий", но займёт меньше времени для скачивания и использует меньше трафика, так как будут скачаны из Интернета только обновившиеся пакеты.

Дополнительные репозитории

Вы можете выбрать дополнительные репозитории, которые будут подключены при установке. Доступны две галочки для официальных репозиториев userland (пакеты, не поставляемые на DVD) и testing (для самых свежих, плохо оттестированных пакетов), а также текстовое поле, в которое вы можете вводить неограниченное число своих собственных репозиториев. Репозитории разделяются переносом строки.
