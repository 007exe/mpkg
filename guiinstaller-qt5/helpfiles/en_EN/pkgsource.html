Package source

Basic settings

There you can choose primary install source and enable additional update.

Install source

There you can choose primary install media:

  • Install from disk (DVD or USB flash): install from media, which was used for booting. If you boot from flash drive with AgiliaLinux install image, also choose this case for installing from flash drive
  • Install from Internet: install from network. All last packages from core32.agilialinux.ru or core64.agilialinux.ru (depending from architecture) will be used to install
  • ISO image on HDD: install from ISO image, which located on some hard disk partition. In this case before next step you must choose path to ISO image. Be careful: partition with install ISO cannot be formated in install process!
  • Directory on HDD: install from folder on HDD, which contains packages. In this case before next step you must choose path to folder with packages!
  • Custom URL: install from custom repository, which can be accessed by URL.

Advanced settings

Automatic update

You can check box "Automatically get updates from official repository" if you want download latest updates from our repository when installs from DVD or ISO. It will look like "Official online repository" but will be faster and takes less Internet traffic because only updated packages will be downloaded.

Additional repositories

You can choose additional repositories which will be used in install process. You can check the box "userland" (enables repo with packages excluded from DVD) and "testing" (enables repo with very last packages, may broke your system), and also can add your custom repo in text field "Manually specified repostiories". Repositories in this field divided by newline.
